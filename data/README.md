
# Data

This directory contains the raw HAM10000 dataset and associated metadata used for training and evaluation.

* `HAM10000_metadata.csv` – Original metadata file with diagnosis labels, patient information, and image identifiers.
* `metadata_clean.csv` – Cleaned and processed metadata after handling missing values and encoding labels.
* `label_mapping.csv` – Mapping from original diagnosis strings to integer class indices.
* `train_split.csv`, `val_split.csv`, `test_split.csv` – CSV files defining the train/validation/test splits used in the notebooks.
