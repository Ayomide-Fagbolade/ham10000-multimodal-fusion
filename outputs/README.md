
# Outputs

The `outputs/` directory contains all generated CSV files from the training and evaluation pipelines. These include:

* **Split definitions** – `train_split.csv`, `val_split.csv`, `test_split.csv`.
* **Metrics** – `test_metrics_efficientnetv2s_image_only.csv`, `test_metrics_fusion_efficientnetv2s_image_only.csv`, `training_history_efficientnetv2s_image_only.csv`, `training_history_efficientnetv2s_metadata_fusion.csv`.
* **Classification reports** – `test_classification_report_efficientnetv2s_image_only.csv`, `test_classification_report_fusion_efficientnetv2s_image_only.csv`.
* **Confusion matrices** – `test_confusion_matrix_efficientnetv2s_image_only.csv`, `test_confusion_matrix_fusion_efficientnetv2s_image_only.csv`.
* **Label mapping** – `label_mapping.csv`.
* **Metadata** – `metadata_clean.csv`.
