# ham10000-multimodal-fusion
Multimodal skin lesion classification on HAM10000, comparing image-only and image-tabular fusion models with class-imbalance handling, GradCAM interpretability, and clinical metadata analysis.

## Folder Structure

| Folder | Description |
|--------|-------------|
| `data/` | Raw dataset, metadata, and image files used for training and evaluation. |
| `figures/` | Visualizations and plots generated during analysis and reporting. |
| `notebooks/` | Jupyter notebooks that walk through data exploration, model training, and evaluation. |
| `outputs/` | CSV files containing split definitions, metrics, confusion matrices, and classification reports for both image-only and multimodal models. |
| `reports/` | Summary reports of model performance and key findings. |
| `src/` | Source code for data processing, model definitions, and training utilities. |
